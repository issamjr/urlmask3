A tool to hide malicious links

## install for linux 
```bash
apt update && apt upgrade
apt install -y git python3
git clone https://github.com/issamjr/urlmask3.git
cd urlmask3
chmod +x urlmask3.py
pip install requests colorama pyshorteners
python3 urlmask3.py
```
